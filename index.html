감정 번역기:
같은 말, 다른 마음

— 귀인이론을 통해 본 메시지 해석의 다양성과 오류 —

같은 상황이나 말도, 사람들은 귀인의 방향에 따라 다르게 해석합니다.
이 차이는 인간관계에서 오해와 갈등을 만들 수 있습니다.

귀인이론이란?

사람들은 타인의 행동이나 메시지를 이해할 때 그 원인을 추론(귀인)하려는 경향이 있습니다.

① 내부 vs 외부 귀인

• 내부: 성격·의도 → ‘그 사람 때문’
• 외부: 상황·환경 → ‘상황 때문’

② 안정 vs 불안정 귀인

• 안정: 지속적/고정적 원인
• 불안정: 일시적/상황적 원인

4가지 귀인 유형 요약

내부 + 안정
“그 애는 원래 나한테 무관심해.”
→ 관계 위협감 증가
내부 + 불안정
“지금 나에게 실망한 것 같아.”
→ 자책·감정적 반응
외부 + 안정
“요즘 바빠서 그래.”
→ 상황 이해
외부 + 불안정
“오늘만 타이밍이 안 맞았어.”
→ 융통성 있음

상황 1: 연애 메시지 해석

0. "4:48-5:20" 를 재생해주세요
1. “내가 매력이 없나?”
2. “내가 말투를 잘못 보냈나?”
3. “원래 무뚝뚝한 사람이야.”
4. “오늘 피곤했겠지.”

해석 유형 설명

내부-안정
자신감 ↓, 표현 감소
내부-불안정
후회 → 행동 수정
외부-안정
짜증·거리두기
외부-불안정
자연스럽게 넘어감

상황 2: 친구 지각

1. “원래 시간 개념이 없지.”
2. “오늘 늦게 준비했나?”
3. “그 지역은 원래 막혀.”
4. “오늘 비 와서 늦었겠지.”

해석 유형 설명

내부-안정
친구 성향 탓
내부-불안정
일시적 실수
외부-안정
환경적 요인
외부-불안정
하루의 변수

상황 3: 교수님 피드백

1. “나는 원래 실력이 부족해서…”
2. “오늘 대충 발표했나?”
3. “교수님 원래 스타일이지.”
4. “오늘 교수님 기분 좋으신가보네.”

해석 유형 설명

내부-안정
위축, 감정적 추리 주의
내부-불안정
개선 행동으로 연결
외부-안정
말투 스타일로 이해
외부-불안정
컨디션 해석
